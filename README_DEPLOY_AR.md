# نشر موقع مكتب القرار الصحيح على GitHub Pages

هذا المجلد جاهز للرفع مباشرة إلى المستودع:

`abdulrhmanalrimi/right_decision`

الرابط المطلوب بعد النشر:

`https://abdulrhmanalrimi.github.io/right_decision/`

## طريقة الرفع السريعة

1. افتح المستودع:
   `https://github.com/abdulrhmanalrimi/right_decision`
2. اضغط `Add file` ثم `Upload files`.
3. ارفع محتويات هذا المجلد نفسها، وليس المجلد كاملًا.
   يجب أن يكون ملف `index.html` في جذر المستودع مباشرة.
4. اضغط `Commit changes` على الفرع `main`.
5. افتح `Settings` ثم `Pages`.
6. في `Build and deployment` اختر:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
7. اضغط `Save`.

بعد اكتمال النشر سيعمل الرابط:

`https://abdulrhmanalrimi.github.io/right_decision/`
