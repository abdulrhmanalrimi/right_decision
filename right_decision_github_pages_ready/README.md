1. ارفع جميع الملفات الموجودة في هذا المجلد إلى مستودع GitHub:

2. تأكد أن الملفات في جذر المستودع مباشرة، وليس داخل مجلد فرعي.

3. من GitHub افتح:
   Settings → Pages

4. اختر Source:
   GitHub Actions

5. احفظ الإعدادات. بعد اكتمال Action سيظهر الموقع على:
   https://abdulrhmanalrimi.github.io/right_decision/
